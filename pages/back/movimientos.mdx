# **Ingresos y Egresos**

Este módulo le permite visualizar los ingresos y egresos de sus tiendas. Podrá visualizar lo siguiente:

- Producto
- Punto de emisión
- Fecha
- Tipo de movimiento
- Cantidad
- Valor total
- Costo / Precio
- Documento (Si es una factura electrónica, se podrá visualizar el documento; si es una compra, se podrá visualizar el número de comprobante)

![Ingresos y Egresos](../../assets/images/movimientosegresosingresos.png)

El módulo de ingresos y egresos cuenta con diferentes opciones para filtrar los movimientos.

### Exportar movimientos

Se pueden exportar los movimientos a un archivo de Excel. Para exportar los movimientos, se debe hacer clic en el botón de exportar.
